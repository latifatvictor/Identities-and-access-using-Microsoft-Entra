# Microsoft Applied Skills – Identity & Access Administrator (Microsoft Entra ID) 
Source: https://learn.microsoft.com/en-us/credentials/applied-skills/get-started-with-identities-and-access-using-microsoft-entra/

## 🎯 Overview
This repository documents my preparation, notes, labs, and hands‑on exercises for the **Microsoft Applied Skills credential: Manage identities and access in Microsoft Entra ID**.

The goal is to build a practical, production‑ready understanding of identity lifecycle management, authentication, access management, and governance capabilities within Microsoft Entra.

---

## 📌 Credential Details
- **Level:** Beginner  
- **Product:** Microsoft Entra ID  
- **Role:** Identity & Access Administrator  
- **Focus Areas:** Identity lifecycle, authentication, access management, identity protection, governance
