# 🔐 Microsoft Entra – User Management Lab (Quick Steps)

## 🧪 Exercise: Perform Basic User Management Tasks

⏱ Duration: ~10 minutes  

---

## 🧩 Task 1 – Create a New User

Step 1 > Go to https://entra.microsoft.com  
Step 2 > Log in to tenant  
Step 3 > Navigate: Users > All users  
Step 4 > Click + New user > Create new user  
Step 5 > Enter user details (UPN, Display Name)  
Step 6 > Copy autogenerated password  
Step 7 > Go to Properties tab  
Step 8 > Enter First Name and Last Name  
Step 9 > Set Usage Location  
Step 10 > Review + Create > Create  

### 🔍 Subtask – Verify Login
Step 1 > Open private browser  
Step 2 > Go to Entra Admin Center  
Step 3 > Log in with new user credentials  
Step 4 > Complete MFA setup  

---

## 🧩 Task 2 – Add a License

Step 1 > Go to https://admin.microsoft.com  
Step 2 > Navigate: Billing > Licenses  
Step 3 > Select a license (e.g. Power Automate Free)  
Step 4 > Click + Assign licenses  
Step 5 > Select user  
Step 6 > Click Assign  

---

## 🧩 Task 3 – Invite External User

Step 1 > Go to Entra Admin Center  
Step 2 > Navigate: Users > All users  
Step 3 > Click + New user  
Step 4 > Select Invite external user  
Step 5 > Enter email, name, message  
Step 6 > Review + Create > Create  

---

## 🧩 Task 4 – Assign Role

Step 1 > Go to Users > All users  
Step 2 > Select user  
Step 3 > Open Assigned roles  
Step 4 > Click + Add assignment  
Step 5 > Select role (e.g. Attribute Definition Reader)  
Step 6 > Choose Assignment type (Eligible)  
Step 7 > Click Assign  

### 🔁 Alternate Method
Step 1 > Go to Identity > Roles & admins  
Step 2 > Select role (e.g. Attribute Log Reader)  
Step 3 > Click + Add assignment  
Step 4 > Select user  
Step 5 > Choose Assignment type (Active)  
Step 6 > Enter justification  
Step 7 > Click Assign  

---

## 🧩 Task 5 – Bulk Import Users

Step 1 > Go to Users > All users  
Step 2 > Click Bulk operations > Bulk create  
Step 3 > Download CSV template  
Step 4 > Edit CSV (update domain, user details)  
Step 5 > Save file  
Step 6 > Upload CSV file  
Step 7 > Click Submit  
Step 8 > Confirm success  
Step 9 > Verify users created  

---

## ✅ Summary Flow

Create User > Assign License > Invite Guest > Assign Role > Bulk Import


# 🔐 Microsoft Entra – User Management (Summary)

## 📌 Overview
Microsoft Entra ID allows you to:
- Create users (internal or external)
- Invite guest users
- Delete users
- Manage access and authentication

---

## 👥 Types of Users

- Internal Member → Employees (full access)
- Internal Guest → Guest access but created in tenant
- External Member → External account + member access
- External Guest → External account + limited (guest) access

✅ Tip:
Choose user type based on access needs

---

## ✅ Prerequisites (Roles)

- Create user → User Administrator  
- Invite guest → Guest Inviter  
- Assign roles → Privileged Role Administrator  

---

## 🧩 Create a New User

Step 1 > Entra > Users  
Step 2 > + New user > Create new user  
Step 3 > Enter:
- Username (UPN)  
- Display name  
- Password  
Step 4 > Set:
- User type (Member/Guest)  
- Location  
Step 5 > (Optional) Add:
- Groups / Roles / Admin unit  
Step 6 > Review + Create  

✅ Result:
User created with defined access  

---

## 🌐 Create External User (External Tenant)

Step 1 > Switch to external tenant  
Step 2 > Users > Create new external user  
Step 3 > Use email as sign-in  
Step 4 > Add properties (optional)  
Step 5 > Create  

✅ Note:
External users authenticate outside your tenant  

---

## 📩 Invite External Guest

Step 1 > Users > Invite external user  
Step 2 > Enter email + display name  
Step 3 > Send invitation  
Step 4 > Guest accepts email invite  

✅ Tip:
Check invitation status (Pending / Accepted)

---

## ➕ Assign Access

During or after creation:

Step 1 > Add to Groups  
Step 2 > Assign Roles (RBAC)  
Step 3 > Assign Licenses  

✅ Best Practice:
Use groups instead of direct assignments  

---

## ❌ Delete a User

Step 1 > Users > Select user  
Step 2 > Click Delete  
Step 3 > Confirm  

✅ Notes:
- User is soft-deleted for 30 days  
- Can be restored within that period  
- Licenses are freed  

---

## 🔁 Lifecycle Summary

Create User > Assign Access > Manage > Disable/Delete  

---

## 🧠 Key Takeaways

- Identity type determines access & authentication  
- External users authenticate differently  
- Use least privilege (RBAC)  
- Always manage lifecycle (Joiner > Mover > Leaver)  
- Deleted users can be restored (30 days)
